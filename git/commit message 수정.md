# git commit --amend
바로 이전의 커밋 메세지만 수정.


| 명령어 | - |
| :--------- | :--------- |
| git commit --amend | 바로 이전의 커밋 <br>메세지만 수정.<br>메세지 수정화면으로 넘어가면,<br> `i` 키를 눌러야 입력 가능.<br> 수정 끝내고 나서 `esc`키 누른 후, `:wq` 입력(수정내용 저장후 수정화면에서 나간다는 명령어)<br> 수정내용 저장안하고 나갈려면 `:q` 명령어 입력 |
